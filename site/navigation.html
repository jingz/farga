Basic Navigation

A Basic horizontal navigation mostly used in top most of the page


  • Home
  • About
  • Service
  • Products

Example Code 
  <nav class="basic">
    <ul>
      <li><a href='#'>Home</a></li>
    </ul>

    <ul>
      <li><a href="#">About</a></li>
      <li><a href="#">Service</a></li>
      <li><a href="#">Products</a></li>
    </ul>
  </nav>
